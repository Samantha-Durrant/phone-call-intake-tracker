⚙️ Version Control Commands

🎨 UI Iteration Commands

Start new iteration:

./ui-iteration.sh start "Description of what you're working on"

Save current changes:

./ui-iteration.sh save "Description of changes made"

Take screenshots:

./capture-screenshots.sh

View iteration history:

./ui-iteration.sh history

📸 Visual Tracking Commands

Get screenshot naming guide:

./ui-iteration.sh visual

View captured screenshots:

./ui-iteration.sh gallery

🔍 Comparison Commands

Compare two versions:

./compare-visuals.sh ui-v1.3-1001 ui-v1.4-1001

List available versions:

./compare-visuals.sh list

🔄 Git Commands

View version history:

git log --oneline

Switch to previous version:

git checkout [version-tag]

Return to current work:

git checkout ui-iterations
